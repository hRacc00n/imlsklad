# IMLSKLAD - Логистический центр

Веб-приложение для управления логистическими операциями.

## Технологии

- **Бэкенд**: Python + Flask
- **Фронтенд**: React + Vite
- **База данных**: SQLite
- **API**: REST

## Установка и запуск

### Бэкенд

```bash
cd backend
python -m venv venv
source venv/Scripts/activate  # Windows Git Bash
# или venv\Scripts\activate  # Windows CMD
pip install -r requirements.txt
python app.py
